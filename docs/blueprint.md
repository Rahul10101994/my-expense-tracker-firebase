# **App Name**: WealthWise

## Core Features:

- Interactive Dashboard: Provides a visual overview of financial status, including income, expenses, and budget summaries.
- Transaction Recording: Enables users to record income and expenses with categorization and tagging.
- Budget Management: Allows users to set and track spending limits for different categories.
- Monthly Financial Summary: Generates detailed monthly reports, breaking down income, expenses, and budget performance. Lets the LLM tool reason when/if to show charts with the summaries.

## Style Guidelines:

- Primary color: Professional blue (#2E86AB) for trust and stability.
- Background color: Light blue-gray (#E0E5EC) for a clean and modern feel.
- Accent color: Calming green (#72A98F) for balance and growth.
- Body and headline font: 'Inter' sans-serif, offering a modern and neutral aesthetic, suitable for both headlines and body text.
- Use a set of simple, consistent icons for transaction categories and dashboard elements.
- Maintain a clean and structured layout with clear sections for each feature.
- Incorporate subtle animations for transitions and feedback on user actions.
